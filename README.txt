SAGULIFE 商談プレゼン / GitHub→Vercel 用

使い方：
1. index.html をリポジトリ直下に置く
2. assets フォルダを作成
3. 画像を以下の名前で assets に配置

assets/slide00.png  表紙
assets/slide01.png  SAGULIFEの対応範囲
assets/slide02.png  特殊加工
assets/slide03.png  01 過去に断った案件、まだ売上に変えられるかもしれません
assets/slide04.png  02 売上拡大は、新規開拓だけではありません
assets/slide05.png  03 断った理由から、強化したい部分が見えてくる
assets/slide06.png  04 SAGULIFEの対応力で、受け皿を広げられるかもしれません
assets/slide07.png  05 見送りやすい案件には、共通点があります
assets/slide08.png  06 小ロットは面倒。でも、次の取引の入口にもなる
assets/slide09.png  07 技術的に難しかった案件も、補完先があれば提案範囲になる
assets/slide10.png  08 キャパ不足で断った案件は、次回の受注候補に変えられる
assets/slide11.png  09 確認したいのは、SAGULIFEが売れるかではありません
assets/slide12.png  10 御社の裏側で、受け皿になれる余地を確認したい

任意：同じ名前の mp4 を置くと自動で動画再生されます。
例：assets/slide03.mp4

操作：
- 発表者画面：通常アクセス
- 顧客表示：右側の「顧客表示を開く」ボタン
- 次へ：Space / →
- 前へ：←
- 全画面：ボタンまたはF（顧客表示）

画像がない場合でも、fallbackスライドが表示されます。
