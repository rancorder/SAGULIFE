SAGULIFE 商談プレゼン Vercel用パッケージ

使い方:
1. このフォルダの index.html と assets フォルダをGitHubへアップロード
2. VercelでGitHubリポジトリをインポート
3. デプロイ後、通常URLは発表者ビュー
4. 「顧客表示を開く」で別ウィンドウを開き、そのウィンドウだけ画面共有

画像ファイル:
assets/page01.png〜page11.png は今回の画像を組み込み済み。
09・10の画像を後から追加する場合は assets/page12.png / assets/page13.png に配置。
MP4動画を使う場合は同名で assets/pageXX.mp4 を置くと自動再生。
